- Экран 1: старт с боли и multi-select.
- Экран 2: цель.
- Экран 3: возраст.
- Экран 4: прошлая форма.
- Экран 5: переходный экран с обещанием.
- Экран 6: колебания веса.
- Экран 7: тест по лестнице.
- Экран 8: шаги.
- Экран 9: стресс и еда.
- Экран 10: переходный экран с отзывом.
- Экран 11: энергия.
- Экран 12: рост.
- Экран 13: текущий вес.
- Экран 14: комфортный вес.
- Экран 15: помощник по питанию.
- Экран 16: доверие.
- Экран 17: время на тренировку.
- Экран 18: частота.
- Экран 19: событие-дедлайн.
- Экран 20: лоадер сбора плана.
- Экран 21: лид-гейт.
- Экран 22: финал-лоадер.
- Экран 23: персональный результат.
- Экран 24: пейволл.

## Визуальная проверка

- На первом экране есть логотип «Усманова Фит» слева сверху.
- Первый экран выглядит как чистый premium hero quiz, без меню, языка и бургер-иконки.
- Hero-визуал не вытесняет 6 карточек ответов слишком низко на 360, 390 и 430 px.
- Карточки ответов крупные, читаемые и нажимаемые.
- Selected state заметный, но не агрессивный.
- Переходные экраны 5, 10, 15, 16, 20 и 22 выглядят как отдельные доверительные блоки, а не как сухой текст.

## Multi-select

- На экране 1 кнопка «Продолжить» disabled без выбора.
- Можно выбрать несколько карточек.
- Повторный клик снимает выбор.
- После выбора кнопка активна.

## Single-choice

- Выбор карточки показывает selected state.
- После короткой задержки происходит переход дальше.
- Нет необходимости нажимать отдельную кнопку.

## Input

- Рост не пропускает пустое значение.
- Рост не пропускает значение вне диапазона 130-220.
- Текущий вес не пропускает значение вне диапазона 35-180.
- Комфортный вес не пропускает значение вне диапазона 35-180.
- Ошибка исчезает после изменения поля.

## Лид-гейт

- Имя обязательно.
- Электронная почта обязательна.
- Электронная почта проверяется на формат.
- Чекбокс согласия обязателен.
- Без согласия результат не открывается.
- Юридические ссылки открываются в новой вкладке.

## LocalStorage

- После успешного лид-гейта появляется запись в `usmanova_quiz_leads`.
- Запись содержит ответы.
- Запись содержит UTM, если они есть в URL.
- Запись содержит `consentAccepted: true`.
- Повторное прохождение добавляет новую запись, а не затирает старую.

## Результат

- Подставляется имя.
- Есть фраза «План собран под ваши ответы».
- Показывается цель.
- Показываются проблемные зоны.
- Показывается текущий вес и комфортный вес.
- Показывается индекс массы тела.
- Показываются калории и КБЖУ.
- Подбираются программы подписки.
- У каждой программы есть пояснение, почему она выбрана.
- Прогноз сформулирован мягко, без обещаний невозможного результата.

## Пейволл

- Есть оффер от результата человека.
- Есть социальные доказательства про Екатерину Усманову.
- Есть блок «до и после».
- Есть тарифы: 390 руб, 1490 руб, 4990 руб.
- Есть состав подписки под цель.
- Есть гарантии и значки безопасности.
- Есть частые вопросы.
- Есть футер с юридическими ссылками.
- Есть место под форму ГетКурса.
- Средний тариф выделен визуально, но без давления.
- Нет таймеров и искусственного дефицита мест.

## Mobile

- 360 px: нет horizontal overflow.
- 390 px: нет horizontal overflow.
- 430 px: нет horizontal overflow.
- Sticky progress не перекрывает контент.
- Sticky CTA не перекрывает важный текст.
- Карточки ответов не вылезают за экран.
- Поля и кнопки удобно нажимать.

## Текстовые правила

- Обращение на «вы».
- В пользовательском интерфейсе нет искусственного дефицита мест.
- Нет неподтверждённых таймеров.
- Нет обещаний невозможного результата.
- Нет тёмной темы.
- Юридические ссылки присутствуют.

- Реальные отзывы и материалы доверия для пейволла.

- Эмбед формы ГетКурса.
